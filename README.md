# Writing Assistant

## Project Idea
This project is a simple writing assistant that analyzes a paragraph of text and provides:
- basic text statistics
- repeated word detection
- long sentence detection
- weak word detection
- synonym suggestions from a small thesaurus

## Files
- `project.py`: main program
- `helpers.py`: helper functions for analysis
- `dictionary.py`: weak words and thesaurus data
- `test_project.py`: tests

## How to Run
```bash
python project.py
